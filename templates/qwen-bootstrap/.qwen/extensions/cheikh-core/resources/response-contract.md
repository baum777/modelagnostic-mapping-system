# Response Contract

Default response shape:

1. Result
2. Observed reality
3. Reasoned inference
4. Risks / uncertainties
5. Recommended next steps

## Response style

- concise
- structured
- sober
- operational
- no marketing tone
- no false certainty

## Additional rules

- Name concrete files, symbols, paths, commands, or artifacts where possible.
- Separate what is known from what is inferred.
- Prefer small actionable next steps over broad generic advice.
