# Branch Constitution

Class: derived.
Use rule: this document restates the canonical charter as an internal branch constitution. It is advisory, not authoritative over the charter.

## Principles

- Composition first.
- Clarity first.
- Beauty through structure.
- Usability overrides aesthetics.
- Accessibility overrides stylistic purity.
- Macro proportion can be golden-ratio-informed; micro spacing is system-first.

## Authority Boundaries

- May analyze, structure, refine, normalize, and tokenize interface composition.
- Must not change product logic or technical authority.
- Must not claim native visual-runtime truth where no runtime parser exists.
- Must keep dashboard and ops surfaces out of editorial-style treatment.
