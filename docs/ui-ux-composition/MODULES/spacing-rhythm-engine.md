# spacing-rhythm-engine

Class: derived.
Use rule: advisory module taxonomy only.

## Purpose

Normalize macro and micro spacing into a coherent rhythm.

## Allowed Modes

- audit
- generate
- refine
- normalize
- tokenize

## Required Inputs

- structure
- spacing tokens
- density preference

## Outputs

- spacing matrix
- rhythm findings
- normalization rules

## Must Not Do

- Collapse all spacing into one scale when distinct rhythms are needed.
