# type-scale-engine

Class: derived.
Use rule: advisory module taxonomy only.

## Purpose

Produce a systemized type scale and line-height scale.

## Allowed Modes

- generate
- normalize
- tokenize

## Required Inputs

- typographic goals
- content density
- device targets

## Outputs

- size scale
- line-height scale
- usage mapping
- breakpoint variants

## Must Not Do

- Create decorative scale jumps with no functional reason.
