Class: derived.

Set or audit typographic order.

## Allowed Modes

- audit
- generate
- refine
- normalize

## Required Inputs

- text system
- content blocks
- brand constraints
- device targets

## Outputs

- type hierarchy map
- readability findings
- typography proposal
- typographic risks

## Must Not Do

- Sacrifice readability for style.
