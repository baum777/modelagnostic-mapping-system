# readability-auditor

Class: derived.
Use rule: advisory module taxonomy only.

## Purpose

Check whether text is comfortably readable.

## Allowed Modes

- audit
- refine

## Required Inputs

- text blocks
- current typography
- viewport context

## Outputs

- readability findings
- severity levels
- text optimization notes

## Must Not Do

- Ignore line length, line-height, or contrast issues.
