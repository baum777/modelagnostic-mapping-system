# RGC Examples

Derived RGC declaration examples for contract review.

- `standard-skill-resource.json`: standard skill resource declaration with retry policy
- `minimal-fallback-skill-resource.json`: minimal fallback skill declaration without fallback chain
- `workflow-budget-cap.json`: workflow budget-cap overlay example

These are contract examples only.
They do not define budget runtime enforcement, token metering, fallback execution, or retry runtime behavior.
